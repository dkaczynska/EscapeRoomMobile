# EscapeRoomMobile
Aplikacja stworzona w ramach pracy dyplomowej

Aplikacja została przygotowana z myślą o urządzeniach mobilnych działających w oparciu o system Android w wersji co najmniej 9. Jest to jedyne ściśle określone wymaganie sprzętowo-systemowe, jakie musi spełniać urządzenie użytkownika.
Instalacja aplikacji odbywa się manualnie, z wykorzystaniem pliku instalacyjnego w formacie APK, który używany jest do instalacji i dystrybucji pakietów w systemie Android. W pierwszym kroku użytkownik powinien pobrać plik EscapeRoomMobile.apk na swoje urządzenie mobilne. Następnie należy przejść do folderu, w którym zapisany został plik, i uruchomić go poprzez wybranie opcji instalacji.
W przypadku pierwszej próby instalacji aplikacji spoza oficjalnego sklepu, system może wyświetlić komunikat z prośbą o zezwolenie na instalowanie aplikacji z nieznanych źródeł. W takiej sytuacji użytkownik powinien wyrazić odpowiednią zgodę 
w ustawieniach systemowych. Po jej udzieleniu możliwe jest ponowne uruchomienie instalatora oraz potwierdzenie chęci instalacji aplikacji. Po zakończeniu procesu instalacji aplikacja jest gotowa do uruchomienia.
